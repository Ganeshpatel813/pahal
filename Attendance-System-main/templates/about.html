About CDGI Attendance System

The CDGI Faculty Attendance System is a production-grade, AI-powered attendance platform built for Chameli Devi Group of Institutions, Indore.

It uses face-api.js (a neural-network face recognition library) to match 128-dimensional facial descriptors. Only the registered faculty member's face can mark their own attendance.

Location is GPS-verified server-side against fixed coordinates of CDGI campus. Attendance is only accepted within 300 metres of the campus — this cannot be spoofed by the browser.

The system tracks check-in and check-out times, computes working hours, auto-detects half-days (<4 hours), and generates monthly reports for both faculty and admin.

Stack: Python Flask · SQLite · face-api.js · GPS Geolocation · Tailwind CSS · bcrypt
